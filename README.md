Calculadora em C# (.NET)
Esta é uma aplicação de console simples desenvolvida em C# que realiza operações matemáticas básicas. O objetivo do projeto foi praticar conceitos de lógica de programação, condicionais (switch/case) e funções locais.

**Funcionalidades**

O programa permite realizar as seguintes operações:

Adição: Soma dois números inteiros.

Subtração: Subtrai o segundo número do primeiro.

Multiplicação: Multiplica dois números inteiros.

Divisão: Divide dois números (suporta casas decimais).

Porcentagem: Calcula a porcentagem de um valor em relação a outro.

Tratamento de Erros: Caso o usuário digite uma opção inválida, o sistema limpa o console e solicita a entrada novamente.

**Como executar o projeto**

Para rodar este projeto em sua máquina local, você precisará ter o SDK do .NET instalado.

Clone o repositório:

Bash

git clone https://github.com/seu-usuario/nome-do-repositorio.git
Acesse a pasta do projeto:

Bash

cd nome-do-repositorio
Execute a aplicação:

Bash

dotnet run
